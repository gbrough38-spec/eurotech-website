# EuroTech of Naples - Modern Website

A modern, responsive website for EuroTech, a premium auto repair and autobody service shop in Naples, Florida.

## Features

- 🚗 **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- ⚡ **Fast & Lightweight** - Pure HTML, CSS, and JavaScript (no heavy frameworks)
- 🎨 **Modern UI** - Clean, professional design with smooth animations
- 📱 **Mobile Menu** - Hamburger menu for mobile devices
- 🔗 **Easy Navigation** - Smooth scrolling between sections
- 📧 **Contact Form** - Direct email integration
- ♿ **Accessible** - Semantic HTML and ARIA attributes

## Sections

1. **Navigation** - Sticky navbar with smooth scrolling
2. **Hero** - Eye-catching banner with call-to-action
3. **Services** - Grid of automotive services
4. **Features** - Why choose EuroTech
5. **Service Menu** - Complete list of available services
6. **About** - Company information
7. **Contact** - Phone, email, location, and contact form
8. **Footer** - Copyright and company info

## How to Use

1. Open `index.html` in your web browser
2. No build process needed - it's ready to go!
3. Customize by editing HTML content and CSS variables

## Customization

- **Colors**: Edit the CSS variables in `styles.css` (`:root` section)
- **Content**: Modify text and service descriptions in `index.html`
- **Contact Info**: Update phone number and email in the contact section
